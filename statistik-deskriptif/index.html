    Statistik Deskriptif¶

    pengertian¶

    statistik deskriptif adalah koefisien penjelas sekilas yang merangkum sebuah set data, yang dapat berupa perwakilan dari seluruh sampel daro populasi. Statistik deskriptif dipecah menjadi metode ukuran pemusatan data dan ukuran simpangan, ukuran pemusatan berisikan mean, median, dan modus, sedangkan ukuran simpangan berisikan standart deviasi, varian, quartil, dan skew

    Tipe Statistik Deskriptif¶

    Mean (rata-rata)¶

    mean adalah nilai rata-rata untuk memperoleh ukuran pemusatan dari data yang dimaksud, mean dapat ditentukan dengan cara menjumlahkan semua nilai data lalu dibagi dengan jumlah data tersebut

    \mathrm{X}=\frac{\mathrm{X}_{1}+\mathrm{X}_{2}+\ldots+\mathrm{X}_{\mathrm{n}}}{\mathrm{n}} \quad \frac{\text { atau } \mathrm{X}=\sum_{i=1}^{n} \mathrm{X}_{\mathrm{i}}}{\mathrm{n}}

    Median (nilai tengah)¶

    Median adalah nilai tengah dari beberapa data, tergantung dari genap ganjilnya jumlah data , jika jumlah datanya ganjil maka nilai media adalah di tengah data-data tersebut sedangkan jika jumlah datanya genap maka median adalah rata-rata dari dua nilai di tengah data-data tersebut.

    1. jika jumlah datanya ganjil $$ M e=X_{\frac{n+1}{2}} $$

    2. jika jumlah datanya genap

    M e=\frac{\frac{X_{n}+X_{n}}{2}+1}{2}

    keterangan :

    Me = Median

    X = nilai data

    n = jumlah data

    Modus¶

    modus adalah nilai suatu data yang sering muncul dalam kumpulan data-data, tidak jarang bisa ditemukan lebih dari satu modus, apalagi bila tidak memiliki banyak data. apabila dalam sekumpulan data terdapat dua modus maka disebut bimodal, jika terdapat tiga modus maka disebut trimodal, dan jika tidak memiliki modus maka sekumpulan data tidak mempunyai modus. $$ \mathrm{Mo}=\mathrm{Tb}+\frac{d 1}{d 1+d 2} \mathrm{c} $$ keterangan :

    Tb = Tepi bawah kelas modus

    d1 = selisih antara frekuensi modus dengan frekuensi sebelumnya

    d2 = selisih antara frekuensi modus dengan frekuensi sesudahnya

    c = panjang kelas

    varian dan standart deviasi¶

    Varian dan standar deviasi (simpangan baku) adalah ukuran-ukuran keragaman (variasi) data statistik yang paling sering digunakan. Standar deviasi (simpangan baku) merupakan akar kuadrat dari varian. $$ s=\sqrt{s^{2}} $$ rumus varian : $$ s^{2}=\frac{n \sum_{i=1}^{n} x_{i}{2}-\left(\sum_{i=1}{n} x_{1}\right)^{2}}{n(n-1)} $$ rumus standart deviasi : $$ s=\sqrt{\frac{n \sum_{i=1}^{n} x_{i}{2}-\left(\sum_{i=1}{n} x_{1}\right)^{2}}{n(n-1)}} $$ keterangan :

    *s*2 = varian

    s = standar deviasi (simpangan baku)

    xi = nilai x ke-i

    n = ukuran sampel

    Quartil¶

    kuartil/quartil adalah nilai-nilai yang membagi data yang sudah diurutkan menjadi 4 bagian yang nilainya sama besar . kuartil pada suatu data didapat dari membagi data tersebutu secara terurut kedalam empat bagian yang memiliki nilai sama besar.

    kuartil terbagi atas tiga macam, yaitu :

    1. kuartil bawah (Q1)
    2. kuartil tengah/median (Q2)
    3. kuartil atas (Q3)
    \begin{array}{l}{Q_{1}=x_{\frac{1}{4}(n+1)}} \\ {Q_{2}=x_{\frac{1}{2}(n+1)}} \\ {Q_{3}=x_{\frac{3}{4}(n+1)}}\end{array}

    Skewness (kemiringan/kecondongan)¶

    skewness dapat dilihat dari perbedaan letak mean, median, dan modusnya. jika ketiga ukuran tersebut berada di titik yang sama maka dikatakan skewness simetris atau distribusi normal. sedangkan jika tidak maka tidak simetris atau tidak distribusi normal $$ \mathrm{Sk}=\frac{\overline{\mathrm{x}}-\mathrm{Mo}}{\mathrm{s}} $$ keterangan :

    Sk = Skewness

    x = rata-rata

    Mo = modulus

    s = Simpangan baku

    Penerapan Statistika Deskriptif dengan Python¶

    untuk menerapkan statistika deskriptif ke dalam python, perlu untuk menginstall beberapa tambahan-tambahan berikut :

    • Text Editor/IDE
    • Library Pandas (pip install pandas)
    • Library Scipy (pip install scipy)

    pertama¶

    untuk bisa memakai Library yang telah diinstall tadi maka perlu dipanggi dengan fungsi import berikut scriptnya :

    import pandas as pd
from scipy import stats

    kedua¶

    lalu memanggil data csv (dibuat sebelumnya), pastikan berada di direktori yang sama agar tidak tersesat

    df = pd.read_csv('tugas.csv', sep=";")

    ketiga¶

    lalu kita buat dictionary untuk menampung nilai-nilai yang ingin ditampikan, lalu kita mengambil data-data dari kolom pada tabel di file .csv yang digunakan dengan cara iterasi setiap barisnya.

    data= {"Statistics": ['Count', 'Max', 'Min', 'Mean','Standard Deviasi', 'Variasi', 'Quantile 1', 'Quantile 2',
                      'Quantile 3', 'Skewnes', 'Median', 'Modus']}
for i in df.columns:
    data[i] = [df[i].count(), df[i].max(), df[i].min(), df[i].mean(), round(df[i].std(),2), round(df[i].var(),2),
               df[i].quantile(0.25), df[i].quantile(0.5), df[i].quantile(0.75), round(df[i].skew(),2), df[i].median(),
              stats.mode(df[i]).mode[0]]

    keempat¶

    tampilkan data tersebut 

    tampilkan = pd.DataFrame(data,columns = ['Statistics'] + [x for x in df.columns])

    mencari outlier¶

    outlier adalah sekumpulan data yang sangat beda sekali dengan sebagian besar nilai lain dalam kelompoknya.

    Biasanya, digunakan formula Z untuk Standardisasi data. $$ z=\frac{(X-\mu)}{\sigma} $$ bisa juga diterapkan pada data yang digunakan tadi dengan menggunakan script ini

    def dekteksi_outlier(df_in):
    outliers = []
    threshold = 3
    for col_name in df_in:
        mean = df_in[col_name].mean()
        std = df_in[col_name].std()
        i=1
        for y in df_in[col_name]:
            z_score = (y - mean) / std
            if abs(z_score) > threshold:
                outliers.append([col_name,y,i])
            i+=1
    return outliers

for i in dekteksi_outlier(df):
    print('Data sampah',i[1],'dikolom',i[0],'pada baris',i[2])
